# spray-deposition-analyzer
雾滴沉积分布分析


主要功能
自动识别 A4 有色背景上的多个贴纸/样片
支持水敏纸、相片纸、铜版纸等多载体
支持手动输入扫描 DPI，或按当前裁剪区域视为完整 A4 自动标定
支持手动删除误检贴纸
支持手动画框补充漏检贴纸
支持覆盖率、斑点数、斑点密度、等效直径、D10/D50/D90、DV0.1/DV0.5/DV0.9 等参数计算
支持结果热图显示与导出
支持多张图像批处理，并导出 ZIP 结果包
批处理导出内容
`combined_results.csv`
`combined_results.xlsx`
每张图的结果 CSV
每张图的结果标注图
每张图的热图 PNG
使用建议
先用自动检测获得大体正确的贴纸位置。
在“手动修正贴纸识别”区域取消误检贴纸的 Keep 选项。
对漏检贴纸使用矩形框进行补充，然后点击“应用当前手动画框”。
对不同扫描分辨率（例如 300/600/1200 dpi）分别上传原始扫描图，程序会基于当前 DPI 换算物理尺寸和雾滴参数。
如果载体上的染色区域发生明显融合，优先解读 Coverage 和热图，粒径相关参数仅作近似参考。



结果表字段说明
程序输出的结果表用于记录每个贴纸区域的自动识别信息、沉积覆盖结果以及雾滴统计参数。各字段含义如下。
1. 区域识别信息
Index
区域序号，对应“检测框与指标”图中的编号。
Label
区域标签。
自动识别区域通常命名为 `R1C1`、`R1C2` 等，表示第几行第几列；手动框选区域通常命名为 `M1`、`M2` 等。
Source
区域来源。
`auto` 表示自动检测得到的贴纸区域，`manual` 表示用户手动框选的区域。
Row
自动识别区域所在行号。手动框选区域该值可能为空或为默认值。
Col
自动识别区域所在列号。手动框选区域该值可能为空或为默认值。
Center_x_px
区域中心点的横坐标，单位为像素（px）。
Center_y_px
区域中心点的纵坐标，单位为像素（px）。
Sticker_area_px
识别出的贴纸区域面积，单位为像素平方（px²）。
---
2. 区域尺寸信息
ROI_height_px
透视矫正并裁剪后区域图像的高度，单位为像素（px）。
ROI_width_px
透视矫正并裁剪后区域图像的宽度，单位为像素（px）。
ROI_area_mm2
区域实际面积，单位为平方毫米（mm²）。
该值根据输入 DPI 或 A4 标定结果换算得到。
---
3. 沉积覆盖结果
Deposit_area_mm2
被识别为雾滴沉积区域的总面积，单位为平方毫米（mm²）。
Coverage_pct
覆盖率，表示沉积区域面积占整个贴纸分析区域面积的百分比，计算方式为：
`Coverage_pct = 沉积面积 / 区域总面积 × 100%`
Coverage_flag
覆盖状态提示。
当覆盖率超过设定的高覆盖率警戒值时，会标记为：
`高覆盖率，可能存在严重重叠`
否则显示为：
`正常`
---
4. 雾滴离散统计参数
以下字段在输出模式选择为 “离散雾滴参数” 时有效。
Spot_count
识别到的独立沉积斑点数量。
Spot_density_per_cm2
单位面积内的斑点数量，单位为个每平方厘米（个/cm²）。
Mean_stain_area_mm2
单个斑点的平均面积，单位为平方毫米（mm²）。
Mean_stain_diameter_um
根据斑点面积计算得到的平均等效斑点直径，单位为微米（μm）。
Mean_output_diameter_um
根据所选直径换算模型修正后的平均输出直径，单位为微米（μm）。
---
5. 直径分布参数
以下参数用于表征按数量统计的斑点或液滴粒径分布特征。
D10_um
10% 的斑点直径小于该值，单位为微米（μm）。
D50_um
中位直径，即 50% 的斑点直径小于该值，单位为微米（μm）。
D90_um
90% 的斑点直径小于该值，单位为微米（μm）。
---
6. 体积分布参数
以下参数用于表征按体积加权后的液滴粒径分布。
DV0.1_um
10% 累积体积对应的液滴直径，单位为微米（μm）。
DV0.5_um
50% 累积体积对应的液滴直径，即体积中位径，单位为微米（μm）。
DV0.9_um
90% 累积体积对应的液滴直径，单位为微米（μm）。
---
7. 检测与分割参数记录
以下字段用于记录本次分析中程序实际采用的关键阈值，便于复现和调参。
Used_board_hue_peak
用于识别蓝色背景纸的主色相峰值。
Used_sticker_value_threshold
用于识别贴纸区域的亮度阈值 `V`。
Used_deposition_threshold
用于分割雾滴沉积区域的颜色距离阈值。
---
说明
当输出模式选择为 “仅覆盖率/染色面积” 时，结果表中通常不包含离散雾滴数量统计及粒径分布相关字段。
当贴纸区域覆盖率过高时，雾滴之间容易发生粘连或重叠，此时单滴识别和粒径统计结果可能偏差较大，应结合 `Coverage_flag` 一并判断。
所有面积和粒径结果均依赖于尺度标定，建议优先使用准确的扫描 DPI。
